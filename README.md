## plugin.video.lasteekraan.err.ee - Kodi addon for [Lasteekraan.err.ee](https://lasteekraan.err.ee)

### 🇪🇪
Kopeeritud yllari [ETV addonist (plugin.video.etv.ee)](https://github.com/yllar/plugin.video.etv.ee) ja kõvasti kohandatud Lasteekraani jaoks Claude'i ja Gemini abiga.

Mõeldud peamiselt omaks tarbeks, mingeid garantiisid ei anna.

#### Märkusi:
* DRM-kaitsega sisu sõltub *libwidevinecdm* versioonist. Vanemate seadmete ja tarkvara puhul (alla **Kodi 21 Omega**) ei pruugi töötada.
* Vajalik on *InputStream Adaptive* olemasolu ja seadistamine.

---

### 🇬🇧
Adapted from yllar's  [ETV addon (plugin.video.etv.ee)](https://github.com/yllar/plugin.video.etv.ee) and heavily modified for Lasteekraan with the assistance of Claude and Gemini.

Intended primarily for personal use, provided as-is with no guarantees.

#### Notes:
* Playback of DRM-protected content depends on the *libwidevinecdm* version. It may not function on versions older than **Kodi 21 (Omega)**.
* Requires the *InputStream Adaptive* addon to be installed and properly configured.
* 🌍 Please note that a lot of non-Estonian content is restricted to Estonian IPs. These items remain listed but will fail to play without a specific error message if accessed from abroad.

---

### Installation / Paigaldamine
1. Download this repository as a ZIP file.
2. In Kodi, go to **Settings** > **Add-ons** > **Install from zip file**.
3. Ensure **InputStream Adaptive** is enabled in your My Add-ons list.
